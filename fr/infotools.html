Projet Infotools

Description du projet :

J'ai réalisé ce projet lors de ma formation en BTS. C'était un projet que j'ai réalisé dans un groupe de 4 personnes, moi compris. J'ai surtout travaillé sur les parties Web et C# étant donné que ce sont les deux parties que j'ai utilisé pour mon dossier de BTS et lors de l'épreuve oral en fin d'année.

Le but de ce projet est de réalisé une application Web, une application lourde C# et une application mobile.

D'abord, la partie Web consiste en un site permettant aux commerciaux de se connecter afin de consulter leurs futurs rendez-vous, la liste des clients ou encore les facturations des achats de ces derniers.

Ensuite, la partie lourde en C# consiste en une application destinée aux administrateurs leur permettant de pouvoir gérer les produits proposés, les commerciaux, les clients et prospects ainsi que d'avoir une vue d'ensemble sur les activités de l'entreprise.

Enfin, la partie application mobile comprend la gestion de RDV par les commerciaux depuis leur téléphone Android.

Exemple d'une page typique de l'application avec tous les champs nécessaires, le tableau avec toutes les données et les différents boutons.
