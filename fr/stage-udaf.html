Stage Udaf

Projet de stage :

J'ai été assigné à un projet de création d'interface personnalisé pour les salariés, celle-ci a pour but de faciliter l'accès aux différents outils nécessaires au travail de chaque utilisateur ainsi que de donner un accès à des informations concernant l'entreprise comme des nouvelles.

Ce projet se dérive en 2 applications, l'application dite "admin" permettant de gérer les différents raccourcis de chacun, d'écrire et de gérer les articles, ainsi que l'application dite "utilisateur".

L'application fonctionne avec une base de données dédiée contenant toutes les informations nécessaires sur les utilisateurs, sur les différents métiers et les articles.

Il est également possible de personnaliser les différents raccourcis en leur ajoutant une icone ou en les positionnant à un endroit spécifique de l'interface.

Concernant les articles, seul le plus récent est affiché sur la page principal mais il est possible de voir les précédents dans un sous-menu dédié.

Ce projet comprend sa documentation technique et utilisateur afin de mieux comprendre le fonctionnement du logiciel et comment l'utiliser.

Exemple d'interface (admin) où l'on peut voir les différents boutons permettant d'ajouter les raccourcis ou de gérer les articles.
